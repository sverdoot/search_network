Welcome to flask-ponywhoosh’s documentation!¶

It’s the easiest way to add a search engine in your flask application on the shoulders of Pony ORM and Whoosh. To prove this, we’ve included some templates to render the search engine. Just checkout the example. Source code and issue tracking at http://github.com/jonaprieto/flask-ponywhoosh.

About

This packages is the missing search engine for your flask application that uses PonyORM as the database manager. It's the easiest way to have a full-text search in your app. Check it out! If you have any feedback or suggestions, let us know.
